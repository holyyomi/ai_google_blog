## 제목 (확정안)

> 세금 환급금 조회 전 홈택스에서 먼저 볼 3가지 (놓치면 못 받는다)

**대안 제목 후보:**
- 홈택스 환급금 조회, 이 3가지 빠뜨리면 환급 못 받습니다
- 국세 환급금 있는데 왜 안 들어왔을까? 홈택스에서 확인할 3가지

---

## 첫 5줄 후킹 오프닝

환급금이 있다고 문자는 왔는데, 통장에는 안 들어온다.
이런 경우 십중팔구 **홈택스에서 막혀 있는 게 있다.**
계좌를 안 등록했거나, 주소가 달라졌거나, 조회 자체를 안 해본 것.
환급금은 신청 안 하면 자동으로 사라지는 돈이 아니지만,
확인을 안 하면 그냥 국고로 다시 돌아간다.

## 사람들이 착각하는 지점 (misconception-box)

| 착각 | 실제 |
|------|------|
| 환급금은 자동으로 통장에 들어온다 | 계좌 미등록 시 자동 지급 안 됨, 국고 귀속 위험 |
| 종합소득세 신고자만 환급 대상 | 근로소득자도 연말정산 이후 환급 발생 가능 |
| 홈택스 환급금 = 연말정산 환급금 | 국세환급금(종합소득세·부가세 포함)은 별도 조회 항목 |
| 문자 왔으면 알아서 입금된다 | 안내 문자 ≠ 자동 지급, 계좌 확인 필수 |

## 실제 기준 (3단계 체크)

### 1단계 — 환급금 존재 확인
- 홈택스(hometax.go.kr) 로그인
- 상단 메뉴 → **[국세환급금 조회]**
- 또는 손택스(모바일) → [조회/발급] → [국세환급금 찾기]

### 2단계 — 계좌 등록 여부 확인
- 홈택스 → [신고/납부] → [환급계좌 개설(변경)]
- 계좌 없으면 입금 불가. 지금 바로 등록 가능

### 3단계 — 주소 일치 여부 확인
- 환급금이 있는데 계좌도 등록했는데 안 들어올 경우
- 주민등록 주소와 홈택스 등록 주소 불일치가 원인인 경우 많음
- 정부24에서 주민등록 주소 확인 후 홈택스 정보 업데이트

## 30초 판단표 (quick-decision-table)

| 내 상황 | 할 일 |
|---------|--------|
| 환급금 조회 안 해봤다 | 홈택스 → 국세환급금 조회 (바로 확인) |
| 조회했는데 금액 없다 | 5월 종합소득세 신고 필요 여부 확인 |
| 금액 있는데 통장 안 들어왔다 | 계좌 등록 → 주소 일치 확인 |
| 계좌도 등록됐는데 아직 안 들어왔다 | 처리 기간 30일 대기 or 세무서 문의 |
| 5년 전 환급금이 있다 | 경정청구 가능 기간인지 확인 |

## 바로 할 행동 3개

1. **홈택스 로그인 후 [국세환급금 조회] 메뉴 클릭** — 금액 있으면 바로 계좌 등록으로 이동
2. **손택스(모바일)에서도 동일 확인 가능** — 앱 설치 후 [조회/발급] 탭
3. **계좌 미등록이라면 [환급계좌 개설(변경)]에서 즉시 등록** — 주거래 통장으로 등록 권장

## FAQ 3개

**Q1. 환급금을 받을 계좌는 어디서 등록하나요?**
홈택스 로그인 → [신고/납부] → [환급계좌 개설(변경)] 메뉴에서 등록합니다. 본인 명의 계좌만 가능합니다.

**Q2. 환급금이 있는데 왜 아직 안 들어왔을까요?**
① 계좌 미등록 ② 주소 불일치 ③ 처리 기간(최대 30일) 중 하나가 원인입니다. 위 3단계 순서대로 확인하세요.

**Q3. 5년 전 환급금도 받을 수 있나요?**
국세기본법상 환급청구 소멸시효는 5년입니다. 경정청구를 통해 신청 가능하지만 세무사 상담을 권장합니다.
